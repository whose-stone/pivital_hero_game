# Datacenter Breach

Isometric maze game built with React + Vite.

## Setup

```bash
npm install
npm run dev
```

## Gameplay

- Explore a dark data center with a flashlight
- Find and fix 4 broken servers (⚡) to collect code fragments
- Use the code to unlock the golden server and recover the drive
- Controls: Arrow keys or WASD to move, Tab for Code Log, Enter to access drives
